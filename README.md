# Daily Restaurant Insights

An end-to-end analytics engineering project powered by Yelp & Google Places API data.

## Structure

- `scripts/` – Python for API ingestion
- `dbt_project/` – dbt models (to be created)
- `docs/` – Documentation (architecture, metrics, changelog)
- `tests/` – Custom dbt tests and macros

## Tech Stack

- BigQuery
- dbt (BigQuery adapter)
- Python
- Docker (for reproducibility)
- Airflow (coming soon)
- GitHub + CI/CD

## Project Goals

- Build a medallion architecture pipeline
- Integrate multiple API sources
- Model versioning, documentation, and stakeholder-ready insights

## Timeline

This project is being built as part of a 21-day learning sprint.
